=> Cherry Pick
Step-by-Step Demo
1️⃣ Create a new folder and initialize Git
mkdir git-cherry-pick-demo
cd git-cherry-pick-demo
git init
2️⃣ Create 5 Java files in master branch

Create these files.

master1.java
public class Master1 {
    public static void main(String[] args) {
        System.out.println("This is master1.java");
    }
}
master2.java
public class Master2 {
    public static void main(String[] args) {
        System.out.println("This is master2.java");
    }
}
master3.java
public class Master3 {
    public static void main(String[] args) {
        System.out.println("This is master3.java");
    }
}
master4.java
public class Master4 {
    public static void main(String[] args) {
        System.out.println("This is master4.java");
    }
}
master5.java
public class Master5 {
    public static void main(String[] args) {
        System.out.println("This is master5.java");
    }
}
3️⃣ Commit each file separately (IMPORTANT)

We must create separate commits so we can cherry-pick the specific file.

git add master1.java
git commit -m "added master1.java"

git add master2.java
git commit -m "added master2.java"

git add master3.java
git commit -m "added master3.java"

git add master4.java
git commit -m "added master4.java"

git add master5.java
git commit -m "added master5.java"
4️⃣ Verify commits
git log --oneline

Example output:

a1b2345 added master5.java
c2d3456 added master4.java
d3e4567 added master3.java
e4f5678 added master2.java
f5g6789 added master1.java

Copy the commit id of master1.java.

Example:

f5g6789
5️⃣ Create an empty branch

Now create a branch without files from master.

git checkout --orphan branch1

Now remove existing files:

git rm -rf .

Check directory:

ls

It should be empty.

6️⃣ Cherry-pick the commit

Now bring only the commit containing master1.java.

git cherry-pick f5g6789
7️⃣ Verify the files
ls

Output:

master1.java

Only master1.java will exist in branch1.

8️⃣ Verify commit history
git log --oneline

You will see only:

f5g6789 added master1.java
